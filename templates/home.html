WE'VE IMPROVED THE ACCURACY OF OBJECT DETECTION USING DATA AUGMENTATION TECHNIQUES

Sharing our Work with Others

This is our latest project. Before embarking on this intriguing journey, we had doubts about finding all the answers. We didn’t know what tools would serve us best, and weren’t sure what resources would be best suited for this type of work. However, months of hard work provided us not only with the answers, but also with new methods to help us explore the things that we're most curious about. We believe that the result speaks for itself - check it out and let us know!

Humans can perceive data using their five senses, similarly machines also need to be able to perceive data, this is enabled in machines with the help of sensory inputs and the traditional input methods like keyboard, mouse, etc. These input methods enable computer vision, computer hearing, and machine touch abilities. With the world being driven by technology towards Automation, it becomes a necessity for machines to be able to take rational decisions with data collected. Humans grasp about 80% of the information with the help of visual ability, similarly, Computer vision plays a vital role when it comes to gathering of data in machines. In this project, we will try to enhance the accuracy of object detection algorithm by using different data augmentation techniques and optimizing the model.
